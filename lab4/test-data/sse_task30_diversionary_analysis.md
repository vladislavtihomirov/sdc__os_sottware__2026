# SSE Task 30: Diversionary Analysis

## Selected system

**Airport luggage tracking and handling system**

## Analysis results

**Undesirable effect:** Bag is sent to the wrong aircraft or destination.

**Harmful scenario:** Sorting system sends the bag to the wrong conveyor belt.

**Harm triangle:** Sorting module + routing data/scanner signal + peak-time baggage flow.

**Protection idea:** Add route-verification checkpoints after sorting to detect and correct misrouting earlier.
