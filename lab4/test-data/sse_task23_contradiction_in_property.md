# SSE Task 23: Contradiction in Property

## System

**Airport luggage system**

## Pair of contradictions in requirements

| Requirement | Description |
|---|---|
| Requirement 1 | The conveyor should move many bags quickly to increase baggage throughput. |
| Requirement 2 | The conveyor should avoid jams, poor spacing, and luggage damage to keep the system reliable. |

---

## Object and property

| Element | Definition |
|---|---|
| Object | Conveyor system |
| Property | Speed of luggage movement |
| Value 1 | High speed |
| Value 2 | Low / controlled speed |s

---

## Contradiction in requirements

**Speed of luggage movement** of **the conveyor system** must be **high**, to **transport many bags quickly and increase airport baggage throughput**, and

**Speed of luggage movement** of **the conveyor system** must be **low / controlled**, to **prevent jams, poor spacing, misrouting, and luggage damage**.
