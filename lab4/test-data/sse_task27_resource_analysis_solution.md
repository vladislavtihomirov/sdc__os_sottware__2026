# SSE Task 27: Resource Analysis & Solution

## Return to the contradiction in property

**System:** Airport luggage system  
**Object:** Conveyor system  
**Property:** Speed of luggage movement

---

## Contradiction in property

The **speed of luggage movement** of the **conveyor system** must be **high** to increase baggage throughput.

The **speed of luggage movement** of the **conveyor system** must be **low / controlled** to prevent jams, poor spacing, misrouting, and luggage damage.

---

## Operation time and space

| Value | Operation time period | Operation space |
|---|---|---|
| **Value 1: High speed** | During normal stable baggage flow | Long straight conveyor sections |
| **Value 2: Low / controlled speed** | During scanning, merging, sorting, congestion, or exception handling | Merge points, scanners, diverters, sorting zones, and loading area entrances |

---

## 1. Substance resources

| Resource | How it can help |
|---|---|
| Existing conveyor belts | Can move bags at different speeds in different zones |
| Bag tags / RFID labels | Can help identify bag priority and route |
| Bags themselves | Can be spaced and grouped according to destination |
| Conveyor gates / stoppers | Can control when bags enter critical zones |
| Guide rails | Can align bags before scanners and diverters |

---

## 2. Field / energy resources

| Resource | How it can help |
|---|---|
| Motor speed control | Can increase or decrease belt speed automatically |
| Sensor signals | Can detect congestion, bag spacing, and position |
| Scanner signals | Can confirm identity before sorting |
| Mechanical vibration | Can separate clustered bags and align luggage |
| Gravity on sloped sections | Can support movement without extra energy |

---

## 3. Information / control resources

| Resource | How it can help |
|---|---|
| Flight schedule data | Can define bag priority and timing |
| Routing database | Can assign the correct conveyor route |
| Bag tracking status | Can show where speed control is needed |
| Congestion prediction algorithm | Can slow down flow before jams happen |
| Real-time control software | Can switch zones between high speed and controlled speed |

---

## Ideal Final Result for selected resources

### Resource 1: Conveyor belt

**Conveyor belt ITSELF creates conditions**  
for **high speed** in **normal stable baggage flow** in **long straight conveyor sections**, and  
for **low / controlled speed** in **scanning, merging, sorting, congestion, or exception handling periods** in **merge points, scanners, diverters, sorting zones, and loading area entrances**.

### Resource 2: Sensor signals

**Sensor signals THEMSELVES create conditions**  
for **high speed** in **normal stable baggage flow** in **long straight conveyor sections**, and  
for **low / controlled speed** in **scanning, merging, sorting, congestion, or exception handling periods** in **merge points, scanners, diverters, sorting zones, and loading area entrances**.

### Resource 3: Real-time control software

**Real-time control software ITSELF creates conditions**  
for **high speed** in **normal stable baggage flow** in **long straight conveyor sections**, and  
for **low / controlled speed** in **scanning, merging, sorting, congestion, or exception handling periods** in **merge points, scanners, diverters, sorting zones, and loading area entrances**.
